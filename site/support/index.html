Support

LocalArchive is open source and does not require an account, login, API key, backend, or paid service.

Report a bug

Open a GitHub issue with the LocalArchive version, Chrome version, operating system, and the page type where the issue occurred.

Privacy questions

Review the privacy policy for the exact local storage, import, export, deletion, and Limited Use disclosures.

Read privacy policy

Known limits

Chrome blocks extensions from injecting into pages such as chrome://, the Chrome Web Store, and some browser-restricted pages.
