LocalArchive

A local-first Chrome extension for saving readable pages and selected text, importing Pocket CSV exports, searching a private archive, and exporting Markdown, HTML, or JSON.

View source Privacy policy

Private by default

Captured pages, selected text, imports, search queries, and exports stay in the user's Chrome profile unless the user exports a file.

Built for migration

Import Pocket CSV files and merge duplicate links by canonical URL, then export selected articles to portable formats.

Open source

The extension source, test instructions, and reviewer notes are public so users and reviewers can inspect the data flow.
